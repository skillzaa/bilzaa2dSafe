Basic Attributes

  1. name

    2. Each Shape has a name. By default this is just "Triangle" , "Rectangle" etc but one can change this name attribute just like any other attribute.

  2. x

    3. This is a x location of the shape. Even incase of a circle there is an x and a y.

  3. y

    4. The y location of the shape.

  4. width

    5. The width of the shape.Incase of a circle it is double its radius.

  5. height

    6. The height of the shape.Incase of a circle it is double its radius.

  6. rotateClockwise

    7. Not implemented yet.

  7. currentRotateAngle

    8. The angle at which the shape should be rotated at the time of draw. This attribute is used to rotate a shape.

  8. color

    9. This refers to the fill color (the background color).

  9. opacity

    10. The opacity range is between 0 (invisible) and 1 (visible)

  10. strokeColor

    11. The color of the border incase is drawn.

  11. shadowColor

    12. The color of shadow. It can be any valid web color value.

  12. shadowBlur

    13. Self evident

  13. shadowOffsetX

    14. Self evident

  14. shadowOffsetY

    15. Self evident
